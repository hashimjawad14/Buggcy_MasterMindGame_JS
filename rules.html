Game Rules

  • 🎯 Objective: Crack the secret code made up of 4 colored pegs in the correct order.
  • 🧩 How to Play:
    • Drag and drop colored pegs from the color palette on the side into the 4 empty slots in the current row.
    • After placing all 4 colors, click the “Check” button to evaluate your guess.
  • 🎯 Feedback Pegs:
    • Black Peg: Correct color in the correct position.
    • White Peg: Correct color but in the wrong position.
    • No Peg: That color does not appear in the code at all.
  • 📌 Code Details:
    • A color can appear only once.
  • 🔁 Game Progression:
    • You have 8 attempts to guess the correct code.
    • The game starts from the bottom row and moves upward one row at a time.
  • 🏆 Winning the Game:
    • Guess all 4 pegs correctly in both color and position to win! 🎉
    • If you don't guess it within 8 turns, the game ends and the code is revealed.
